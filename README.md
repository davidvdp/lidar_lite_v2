# lidar_lite_v2
Python module for lidar lite version 2 distance measurements.

## Installation
From lidar_lite_v2 directory:

```$ pip3 install .```

## Execution of module
```$ python3 -m lidar_lite_v2```

## Python import
```
>>> from lidar_lite_v2 import LidarLiteV2
>>> llv2 = LidarLiteV2()
>>> llv2.get_distance()
```
